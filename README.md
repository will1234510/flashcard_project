# Flashcard_Project

A flashcard application designed for efficient study and memorization.

**Technologies Used:**
- **Frontend:** React.js, Tailwind CSS
- **Routing:** React Router DOM
- **State Management:** React Context API
- **Storage:** Local Storage (will change soon)

**Key Features:**
- **Folder Organization:** Create, manage, and delete folders to organize study sets.
- **Interactive Flashcards:** 3D flip animations for terms and definitions.
- **Media Support:** Support for both text and image-based flashcards (including copy-paste image support).
- **Responsive Design:** Fully responsive layout with a collapsible sidebar and mobile-friendly navigation.
